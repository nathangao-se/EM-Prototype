CSV Table Selection Experiment

How to use: 1. Upload a CSV file or load sample data
2. Click any data cell to select it; click + drag to select a range
3. Click a row number to select a row; drag across row numbers for multiple rows
4. Click a column header to select a column; drag across headers for multiple columns
5. Hold Shift and click to extend any selection
6. Use Save Block to capture the selection, or select an entire row and use Save with Header to use it as a header with all continuous rows below as the body
